GET api/entities/lookup/feed

  • Description
  • Parameters
  • Results
  • Examples

Description

Parameters

* path path of the instance for which you want to determine the feed. e.g. /data/project1/2014/10/10/23/ Path has to be the complete path and can't be a part of it. * doAs <optional query param> allows the current user to impersonate the user passed in doAs when interacting with the Falcon system.

Results

Returns the name of the feed along with the location type(meta/data/stats) and cluster on which the given path belongs to this feed.

Examples

Rest Call

GET http://localhost:15000/api/entities/lookup/feed?path=/data/project1/2014/10/10/23&doAs=joe

Result

{ "feeds": [ { "feedName": "My-Feed1", "locationType": "DATA", "clusterName": "My-cluster1" }, { "feedName": "My-Feed2", "locationType": "DATA", "clusterName": "My-cluster2" } ] }
